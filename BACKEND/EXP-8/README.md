About This Project

This is a simple backend project made using Flask.
In this project, I created basic API routes to manage student data.

The main aim of this experiment is to understand how backend works and how to structure a Flask project properly.

What I Learned

How to create a Flask app

How to use Blueprints

How to create API routes

How to return JSON responses

How to prepare a project for deployment

How to Run the Project

Go to the EXP-8 folder

Create a virtual environment

Install requirements

Run the server

Run using:

python run.py

The server will run on:

http://127.0.0.1:5000
Sample Route

When we open:

GET /

It returns:

{
  "message": "Backend Server is running"
}
Conclusion

This experiment helped me understand backend development in a simple and practical way.
