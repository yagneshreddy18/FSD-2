# React + Vite

This project is built using **React** with **Vite** as the build tool.  
Vite provides a fast development environment with Hot Module Replacement (HMR) and an optimized production build.

The project follows modern frontend practices while keeping the structure simple and easy to understand.

---

## Project Features

- Fast development using Vite
- Component-based UI with React
- Hot Module Replacement (HMR)
- Optimized production build
- Ready for deployment on Netlify

---

## Learning Outcomes

After completing this project, the learner will be able to:

1. Understand the basic structure of a React application created using Vite.
2. Build and run a React project using modern build tools.
3. Differentiate between development mode and production build.
4. Deploy a React application on Netlify using the `dist` folder.
5. Understand the working concept of Single Page Applications (SPA).

---

## Difference Between SPA and MPA

| Feature | SPA (Single Page Application) | MPA (Multi Page Application) |
|------|------------------------------|------------------------------|
| Page Load | Loads a single HTML page | Loads a new HTML page for every request |
| Speed | Faster after initial load | Slower due to repeated page loads |
| Navigation | Happens without page reload | Page reloads on navigation |
| User Experience | Smooth and app-like | Traditional website behavior |
| Example | React, Angular, Vue apps | Classic HTML/PHP websites |
