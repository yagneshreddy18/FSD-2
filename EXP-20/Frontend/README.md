# Testing Frontend

## Description

This frontend project contains a simple React login form built with Vite and Material UI. It is used as the frontend testing example for the unit testing experiment.

---

## Features

1. Login form with email and password fields
2. Basic client-side validation for required inputs
3. Material UI components for form controls
4. Component tests written with Vitest and Testing Library

---

## Run the Project

```bash
npm install
npm run dev
```

---

## Run Tests

```bash
npm run test
```

---

## Outcome

The project demonstrates how a small React form can be tested reliably using Vitest.
